Quick answers about using Toolker. Open a question below, or browse PDF tools, image tools, text tools, and generators from the home page.

Using Toolker

Is Toolker free?

Yes. The tools on this site are free to use for typical personal and work tasks. We aim to keep core features accessible without a subscription.

Do I need to sign up?

No. There is no account or login. Open a tool, use it, and you are done.

How do the tools work?

Most tools run in your web browser: you upload a file or paste text, the page processes it, and you download or copy the result. Each tool page explains what it accepts (for example PDF-only or JPG/PNG). Pick a category to get started: PDF, images, text, or generators.

Trust & safety

Are my files stored on your servers?

For tools that work entirely in your browser, files are not stored after processing. See the Privacy Policy for the full picture, including features that call external APIs (like live rates) without storing your documents.

Is it safe to upload files?

When a tool runs locally in your browser, files are not stored after processing. Use a trusted network and browser, and avoid sharing sensitive files on shared computers. If you are unsure, read how we handle data and our Terms of Use.

What does Toolker do with my data?

We do not build a personal dossier from your files. Local tools process data in memory and discard it when you are done. We may use minimal technical data (such as theme preference) to make the site work smoothly—details are in the Privacy Policy.

File limits & performance

Are there file size limits?

Practical limits depend on your device memory and browser. Very large PDFs or images may run slowly or fail on older hardware. Try a smaller file or a different browser if something stalls.

Why is my file processing slowly?

Large files, many pages, and slower devices all add time. Closing other heavy tabs and trying again often helps.

Troubleshooting

My file did not work. What should I check?

Confirm the format matches what that tool expects, that the file is not corrupted, and that it is not unusually large. Refresh the page and try again.

The page or tool seems stuck.

Reload the page, try another modern browser (Chrome, Firefox, Edge, Safari), or restart the browser if memory is low.
